# Shengning Wang - Academic Homepage

This repository contains the source for [sn-wang.github.io](https://sn-wang.github.io), built with Jekyll and hosted on GitHub Pages.

## Content map

- `_pages/about.md`: homepage biography and research goal
- `_publications/`: publication records
- `_projects/`: publicly approved research projects
- `_portfolio/`: research prototypes and media demonstrations
- `_talks/`: talks and presentations
- `_posts/`: blog posts
- `images/`: portrait and image assets
- `files/`: CV and portfolio video assets
- `_config.yml`: site-wide identity and build settings
- `_data/navigation.yml`: top navigation

## Local preview

Install the dependencies once, then build the site:

```bash
bundle install
bundle exec jekyll build
```

For a local preview:

```bash
bundle exec jekyll serve -l -H 127.0.0.1
```

The site retains the AcademicPages and Minimal Mistakes theme attribution in the footer and license files.
